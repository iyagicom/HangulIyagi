# HangulIyagi 노린 (Norin) v1.4.0

노린(Norin)은 순수 우리말의 의미를 담은 독자적인 한글 입력기다.  
기존 입력기와 전혀 다른 구조로 설계된 새로운 입력 시스템이다.

외부 입력 프레임워크에 의존하지 않으며, 자체 구현된 입력 엔진으로 동작한다.

- HangulIyagi는 Linux 한글 입력의 새로운 구조를 완성했다.  
- 이제 한글 입력은 HangulIyagi 노린으로 날개를 달았다.

---

## 개발 배경

기존 한글 입력기는 조합 상태 기반 구조로 인해  
포커스 이동, 마우스 클릭 등에서 마지막 글자가 사라지는 문제가 존재한다.

이는 구조적인 한계이며 오랫동안 해결되지 않았다.

HangulIyagi는 이 문제를 해결하기 위해  
처음부터 새로운 입력 구조로 설계되었다.

👉 어떤 상황에서도 입력 중인 글자가 소실되지 않도록 설계됨

## 지원 환경

- **Ubuntu 22.04 / 24.04** — 완전 테스트 완료
- **Fedora / openSUSE / Arch** 등 — install.sh 지원, 동작 여부 미검증

번들 Qt5를 사용하는 서드파티 앱(GStarCAD 등)에도 플러그인 직접 연결 가능.  
Qt 버전별로 독립 빌드된 플러그인 제공 (`libhanguliyagi-qt5.14.so` 등).

## 기능

- 두벌식 한글 입력 (자체 엔진, libibus 없음)
- 한/영 전환 (한/영키, Shift+Space, Ctrl+Shift, 사용자 정의 키)
- 쌍자음 입력 (Shift+자음) — ㅃㅉㄸㄲㅆ은 CapsLock과 무관, Shift로만 입력
- **영→한 자동변환** — 영문 모드에서 타이핑 후 한/영 전환 시 자동으로 한글로 변환
- **시작 모드 선택** — 영문 시작 / 한글 시작 / 이전 상태 유지 (앱별 공유)
- 포커스 변경 시 자동 입력 확정
- 빠른 키 단위 삭제
- 단어 후보 창 (타이핑 중 자동 완성)
- 영문 모드에서도 후보 단어 표시
- 사용자 사전 자동 학습 (설정 글자 수 이상 단어)
- 사용자 사전 SQLite 저장 (`~/.config/IYAGI-INC/HangulIyagi/hanguliyagi.db`)
- 관리자 창 (`hanguliyagi-manager`)

## 설치

### 방법 1. .deb 패키지 (Ubuntu / Debian 계열)

`hanguliyagi_1.4.0_amd64.deb` 파일을 **파일 관리자에서 더블클릭**하면 설치된다.

설치 후 입력기 선택:

1. **설정 → 시스템 → 언어 및 지원 → 설치된 언어 관리** 열기
2. 하단 **키보드 입력기** 항목에서 **HangulIyagi** 선택
3. 리부팅 (로그아웃 후 다시 로그인 안될 가능성 확인됨)

> 설치 후 **한글이야기** 앱을 실행하면 후보 단어 창, 입력 방식 등  
> 세부 설정을 확인하고 조정할 수 있다.

---

### 방법 2. ZIP (Fedora, Arch, openSUSE 등)

```bash
unzip hangulIyagi-v1.4.0-linux-x64.zip
cd hangulIyagi-v1.4.0-linux-x64
./install.sh
```

`im-config`가 없는 배포판(Fedora, Arch 등)에서는 install.sh가 자동으로  
환경변수를 설정합니다:

- **Wayland**: `~/.config/environment.d/90-hanguliyagi.conf`
- **X11**: `~/.xprofile`

로그아웃 후 다시 로그인.

> Ubuntu/Debian 외 배포판은 동작 여부가 미검증입니다.

---

### Qt5 앱에 직접 연결 (선택, Qt 자체 방식)

Qt 자체 방식을 선택하거나, Qt5 앱 실행 스크립트에서 직접 설정:

```bash
export QT_IM_MODULE=hanguliyagi
# XIM 이중 입력 방지
unset XMODIFIERS
```

Qt5 버전별 플러그인을 앱의 `platforminputcontexts/` 디렉토리에 설치:

```bash
# Qt5.14 앱 예시 (GStarCAD 2026 등)
cp libhanguliyagi-qt5.14.so /앱경로/qtplugins/platforminputcontexts/libhanguliyagi.so
```

## 변경 이력

### v1.4.0
- **영→한 자동변환**: 영문 모드에서 타이핑 후 한/영 전환 시 자동으로 한글로 변환 (관리자에서 ON/OFF)
- **시작 모드**: 영문 시작 / 한글 시작 / 이전 상태 유지 3가지 선택 (관리자 → 키 설정)
- 이전 상태 유지 모드 — 모든 앱이 D-Bus 신호로 마지막 한/영 상태를 공유
- **CapsLock 버그 수정**: 한글 모드에서 CapsLock이 쌍자음(ㅃㅉㄸㄲㅆ)을 잘못 활성화하던 문제 해결 (GTK3/Qt6/D-Bus/XIM 전체)
